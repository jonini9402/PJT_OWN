# PJT_OWN

> 본 프로젝트는 2인 팀 프로젝트입니다.  
> 사용자 및 콘텐츠 데이터를 관리하는 웹 서비스로,  
> 백엔드 REST API 일부 설계·구현과 프론트엔드 화면 일부를 담당했습니다.
>
>
> 원본 Repo: https://github.com/echo98100/PJT_OWN

## 프로젝트 개요
- Java·Spring 기반 웹 서비스
- 사용자 및 콘텐츠 데이터를 관리하는 백엔드 시스템 구현
- REST API 중심의 서버 구조 설계

## 주요 기능
- 사용자 계정 관리 API (등록, 조회)
- 음악 및 운동 기록 데이터 CRUD API
- 조건별 데이터 조회 API
- 공통 응답 구조 및 예외 처리

## 담당 역할
- 사용자 및 콘텐츠 데이터 관리 REST API 설계·구현
- DB 테이블 설계 및 JPA 기반 연동
- 조건별 조회 및 데이터 검증을 포함한 비즈니스 로직 구현
